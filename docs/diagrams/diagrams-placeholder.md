# 다이어그램 보관 위치 (PlantUML · Mermaid)

본 디렉토리는 시스템 UML 산출물을 텍스트 소스 형태로 보관한다.

작성 예정 산출물:

1. `usecases.puml` — 사용 사례 다이어그램
2. `seq-*.puml` — 주요 시나리오 시퀀스 다이어그램
3. `agent-graph.mmd` — 추천 에이전트 노드 그래프 (Mermaid)
4. `c4-system.puml` — C4 컨테이너 · 배포 통합 다이어그램
5. `class-*.puml` — 핵심 도메인 클래스 다이어그램

본 파일은 자리표시이며, 실제 다이어그램 소스가 추가되면 대체된다.
